Yetkili Erişim Gereklidir

Bu API uç noktaları (endpoints) sadece kurumsal müşterilerimiz içindir. Aşağıdaki kodlar örnektir. Gerçek bağlantı için lütfen bizimle iletişime geçerek API SECRET KEY talep ediniz.

API Entegrasyonu v2.0

Botlarımızı kendi sistemlerinize entegre etmek için aşağıdaki yapıyı kullanabilirsiniz.

Python ile Bağlantı Örneği

import requests # UYARI: Size verilen özel anahtarı buraya girin. API_KEY = "NK_MUSTERI_OZEL_ANAHTARI_BURAYA" BASE_URL = "https://api.nurullahkartal.com.tr/v1" def bot_durumu(): headers = { "Authorization": f"Bearer {API_KEY}" } response = requests.get(f"{BASE_URL}/status", headers=headers) if response.status_code == 200: print("Sistem Aktif:", response.json()) else: print("Yetkisiz Erişim!") bot_durumu()

Hata Kodları

200 OK: İşlem başarılı.

401 Unauthorized: Geçersiz veya eksik API anahtarı.

429 Too Many Requests: İstek limiti aşıldı.
