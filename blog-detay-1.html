Python ile 10 Dakikada Telegram Botu Nasıl Kodlanır?

Nurullah Kartal 12 Aralık 2025

Telegram botları, işletmelerin müşteri ilişkilerini otomatize etmesi veya kişisel projelerinizde bildirimler almanız için harika araçlardır. Bu rehberde, Python ve telebot kütüphanesini kullanarak basit bir "Merhaba Dünya" botu yapacağız.

1. BotFather ile Bot Oluşturma

İlk adım olarak Telegram uygulamasını açın ve arama çubuğuna @BotFather yazın. Başlat dedikten sonra şu komutu gönderin:

/newbot

BotFather size botunuzun adını ve kullanıcı adını soracak. İşlem bittiğinde size uzun bir API TOKEN verecek. Bunu saklayın.

2. Gerekli Kütüphanenin Kurulumu

Bilgisayarınızın terminalini (CMD) açın ve şu komutu yazın:

pip install pyTelegramBotAPI

3. Python Kodlarını Yazalım

Şimdi favori kod editörünüzü (VS Code) açın ve bot.py adında bir dosya oluşturup şu kodları yapıştırın:

import telebot # BotFather'dan aldığınız tokeni buraya yapıştırın API_TOKEN = 'BURAYA_TOKEN_GELECEK' bot = telebot.TeleBot(API_TOKEN) # /start komutuna cevap ver @bot.message_handler(commands=['start']) def send_welcome(message): bot.reply_to(message, "Merhaba! Ben NKartal tarafından kodlandım.") # Mesajları dinlemeye başla print("Bot çalışıyor...") bot.infinity_polling()

Sonuç

Terminalden python bot.py komutunu çalıştırın. Botunuz artık aktif! Telegram'dan botunuza mesaj attığınızda size cevap verecektir.

Daha gelişmiş bot projeleri ve kurumsal otomasyon çözümleri için iletişime geçebilirsiniz.
